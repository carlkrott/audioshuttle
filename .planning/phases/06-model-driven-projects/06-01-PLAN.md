**1. GENRE_PROFILES** — dict keyed by genre name (lowercase). Each profile:
```python
{
    "tempo_range": (min, max),     # BPM range for the genre
    "default_tempo": int,          # default BPM when user doesn't specify

}
```

Include these genres at minimum:
- **rock**: 100-140 BPM (default 120), sections: intro(4)-verse(16)-chorus(8)-verse(16)-chorus(8)-solo(16)-chorus(8)-outro(8), instruments: drums, bass, rhythm_guitar, lead_guitar, keys, vocals
- **pop**: 90-130 BPM (default 120), sections: intro(4)-verse(16)-prechorus(4)-chorus(8)-verse(16)-prechorus(4)-chorus(8)-bridge(8)-chorus(8)-outro(8), instruments: drums, bass, keys, pad, melody, vocals
- **electronic**: 120-140 BPM (default 128), sections: intro(8)-buildup(16)-drop(8)-breakdown(8)-buildup(16)-drop(8)-outro(8), instruments: drums, sub, bass, synth, pad, arp, fx

- **jazz**: 80-160 BPM (default 120), feel="swing", sections: intro(8)-head(32)-solo(32)-head(32)-outro(8), instruments: drums, upright_bass, piano, melody, pad
- **orchestral**: 60-120 BPM (default 90), sections: intro(4)-a(16)-b(16)-a(16)-climax(16)-outro(8), instruments: strings, brass, woodwinds, percussion, harp, timpani
- **ambient**: 60-90 BPM (default 70), sections: intro(16)-a(32)-b(32)-a(32)-outro(16), instruments: pad, strings, keys, fx, melody
- **metal**: 100-200 BPM (default 160), sections: intro(4)-verse(16)-chorus(8)-verse(16)-chorus(8)-solo(16)-breakdown(8)-chorus(8)-outro(4), instruments: drums, bass, rhythm_guitar, lead_guitar, keys
- **reggae**: 60-90 BPM (default 75), feel="half-time", sections: intro(4)-verse(16)-chorus(8)-verse(16)-chorus(8)-bridge(8)-chorus(8)-outro(8), instruments: drums, bass, rhythm_guitar, keys, melody
- **funk**: 90-120 BPM (default 110), sections: intro(4)-verse(16)-chorus(8)-verse(16)-chorus(8)-bridge(8)-chorus(8)-outro(8), instruments: drums, bass, rhythm_guitar, keys, horns, melody
- **blues**: 60-120 BPM (default 100), feel="shuffle", sections: intro(4)-verse(12)-verse(12)-solo(12)-verse(12)-outro(4), instruments: drums, bass, rhythm_guitar, lead_guitar, keys

**2. INSTRUMENT_FAMILIES** — dict mapping instrument roles to family groups for bus routing:
```python
INSTRUMENT_FAMILIES = {
    "guitars": {"rhythm_guitar", "lead_guitar", "guitar"},
    "strings": {"strings", "violin", "viola", "cello", "string", "orchestral_strings"},
    "brass": {"brass", "trumpet", "trombone", "horns", "saxophone"},
    "woodwinds": {"woodwinds", "flute", "clarinet", "oboe"},
    "vocals": {"vocals", "vocal", "voice", "choir"},
    "synths": {"synth", "pad", "arp", "fx", "riser", "keys", "key"},
    "percussion": {"drums", "drum", "beat", "kick", "snare", "percussion", "timpani"},
}
```

**3. FX_CHAINS** — dict keyed by instrument family, each containing per-genre FX pipeline:
```python
FX_CHAINS = {
    "guitars": {
        "rock": [
            {"plugin": "ReaEQ", "preset": None},
            {"plugin": "ReaComp", "preset": None},
        ],
        "metal": [
            {"plugin": "ReaEQ", "preset": None},
            {"plugin": "ReaComp", "preset": None},
            {"plugin": "ReaGate", "preset": None},
        ],
        # generic fallback for unspecified genres
        "_default": [
            {"plugin": "ReaEQ", "preset": None},
            {"plugin": "ReaComp", "preset": None},
        ],
    },
    "drums": {
        "rock": [
            {"plugin": "ReaComp", "preset": None},
            {"plugin": "ReaEQ", "preset": None},
        ],
        "electronic": [
            {"plugin": "ReaComp", "preset": None},
            {"plugin": "ReaEQ", "preset": None},
            {"plugin": "ReaLimit", "preset": None},
        ],
        "_default": [
            {"plugin": "ReaComp", "preset": None},
        ],
    },
    # ... for every family in INSTRUMENT_FAMILIES
}
```

**Helper functions:**
- `get_genre(genre_name: str) -> dict` — case-insensitive lookup with fallback to "rock" default
- `get_family(role: str) -> str` — map any instrument role to its family (e.g. "lead_guitar" → "guitars")
- `get_fx_chain(role: str, genre: str) -> list[dict]` — resolve FX chain for an instrument in a genre, falling back through families and defaults
- `get_tempo(genre_name: str, user_tempo: int | None = None) -> int` — use user tempo if provided, otherwise genre default, otherwise 120
- `validate_profile(genre_name: str) -> bool` — check a profile has all required fields

Use ALL_CAPS for constants, snake_case for functions. No class — this is a data module. Use `from __future__ import annotations` at top.
