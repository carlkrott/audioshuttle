  <action>
    **config.py** — Pydantic Settings class for all configuration:

    ```python
    from pydantic_settings import BaseSettings

    class Settings(BaseSettings):
        # Reaper OSC connection
        reaper_host: str = "127.0.0.1"
        reaper_port: int = 8000          # Reaper listens here
        reaper_feedback_port: int = 9000 # Reaper sends feedback here

        # Embedded model
        model_api_url: str = "http://localhost:8092/v1/chat/completions"
        model_name: str = "gemma-4-e2b"

        # Chat interface (external AI example)
        chat_api_url: str = "http://localhost:8090/v1/chat/completions"
        chat_model_name: str = "gemma-4-e4b"

        # Server
        web_host: str = "127.0.0.1"
        web_port: int = 8765
        memory_vault_path: str = "~/.audioshuttle/memory"

        model_config = {"env_prefix": "AUDIOSHUTTLE_"}
    ```

    **models.py** — Pydantic models for the domain:

    - `TrackState`: track_number (int), name (str), volume (float 0-1), pan (float -1 to 1), mute (bool), solo (bool), selected (bool)
    - `TransportState`: playing (bool), recording (bool), position_seconds (float), tempo (float), time_signature (str)
    - `OSCCommand`: address (str), args (list[float|int|str]), description (str — human readable)
    - `CommandResult`: success (bool), address (str), sent_value (any), reaper_feedback (any — optional), error (str — optional)
    - `DAWState`: tracks (list[TrackState]), transport (TransportState), project_name (str), timestamp (float)

    All models should use `model_config = {"populate_by_name": True}` for serialization flexibility.
  </action>
  <verify>

    t = TrackState(track_number=1, name='Drums', volume=0.75, pan=0.0, mute=False, solo=False, selected=False)
    print(f'Track: {t.name} vol={t.volume}')
    cmd = OSCCommand(address='/track/1/volume', args=[0.75], description='Set track 1 volume to 75%')
