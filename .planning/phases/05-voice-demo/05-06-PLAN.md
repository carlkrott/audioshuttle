**1. Header:**
- "AudioShuttle" with tagline: "Gemma enables the walk-in recording studio — speak to create"
- Badges: Python 3.12+, MIT License, Tests
- One-paragraph hook: Gemma's lightweight efficiency bridges the gap between human speech and professional audio production, making music creation accessible to anyone who can speak.

**2. 🎯 The Problem:**
- Professional audio production requires complex mouse/keyboard workflows
- Musicians with motor disabilities, RSI, or visual impairments are locked out
- "Imagine walking into a recording studio and just... speaking. That's what AudioShuttle enables."

**3. 💡 The Solution — Powered by Gemma:**
- Gemma E2B as the translation layer between human speech and DAW commands
- Two-brain architecture: External AI (any LLM) = creative producer, Gemma E2B = precise audio engineer
- Whisper converts speech → text → Gemma translates → OSC commands → Reaper executes
- "The first step toward a literal walk-in recording studio where anyone who can speak can produce music"

**4. 🏗 Architecture:**
- ASCII diagram: `User → Speech/Text → [Whisper] → [Gemma E2B] → [OSC] → Reaper DAW`
- Explain each component
- 20 MCP tools for precise DAW control

**5. 🚀 Quick Start:**
```bash
git clone https://github.com/user/audioshuttle && cd audioshuttle
python -m venv .venv && source .venv/bin/activate
pip install -e ".[all]"

# Start Reaper, then:
audioshuttle                    # Standalone: web UI + tray
audioshuttle --transport stdio  # MCP server mode (for Gemini CLI, Claude, etc.)
```

Create `.github/workflows/ci.yml`:
```yaml
name: Tests
on: [push, pull_request]
jobs:
  test:
    runs-on: ubuntu-latest
    strategy:
      matrix:
        python-version: ["3.12", "3.13", "3.14"]
    steps:
      - uses: actions/checkout@v4
      - name: Set up Python ${{ matrix.python-version }}
        uses: actions/setup-python@v5
        with:
          python-version: ${{ matrix.python-version }}
      - name: Install dependencies
        run: |
          python -m pip install --upgrade pip
          pip install -e ".[web,tray]"
          pip install pytest pytest-asyncio
      - name: Run tests
        run: pytest --tb=short -q
```
