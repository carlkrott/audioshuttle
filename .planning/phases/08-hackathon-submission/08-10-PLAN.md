### Step 2: Compile final video with ffmpeg

Create a concat file listing all sources in order:

```
file 'out/intro.mp4'
file 'demo-video/scene-01-hook.mp4'
file 'demo-video/scene-02-gemini-cli.mp4'
file 'demo-video/scene-03-voice.mp4'
file 'demo-video/scene-04-project-gen.mp4'
file 'demo-video/scene-05-impact.mp4'
file 'out/outro.mp4'
```

Then compose with BGM:

```bash
ffmpeg -f concat -safe 0 -i concat.txt \
  -i remotion/public/audio/bgm-hackathon-loop.wav \
  -filter_complex "[1:a]volume=0.15,adelay=5000|5000[a]" \
  -map 0:v -map "[a]" -c:v libx264 -crf 18 -preset slow \
  -c:a aac -b:a 192k -shortest \
  demo-video/final.mp4
```

Key mixing choices:
- BGM at 15% volume (supports voiceover, not distracts)
- BGM starts 5 seconds in (after intro, when voiceover begins)
- BGM loops if final video exceeds 30s
- Video at CRF 18 (near-lossless, keeps text readable)
- AAC audio at 192k (good for voice clarity)
