MCP Server Configuration

Add AudioShuttle's MCP server to your AI client so it can control your DAW. Copy the JSON for your tool:

Claude Code / Claude Desktop 
{
  "mcpServers": {
    "audioshuttle": {
      "command": "python",
      "args": ["-m", "audioshuttle", "--transport", "stdio"]
    }
  }
}

Add to ~/.claude/settings.json or Claude Desktop's claude_desktop_config.json

Gemini CLI 
{
  "mcpServers": {
    "audioshuttle": {
      "command": "python",
      "args": ["-m", "audioshuttle", "--transport", "stdio"]
    }
  }
}

Add to ~/.gemini/settings.json

SSE Client (Cursor, n8n, etc.) 
{
  "mcpServers": {
    "audioshuttle": {
      "url": "http://localhost:8765/mcp/sse"
    }
  }
}
